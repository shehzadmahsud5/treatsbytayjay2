# Treats By Tayjah - Website Implementation Workflow

This document outlines the step-by-step workflow to build a simple, beautiful, and highly effective website for Treats By Tayjah. It defines exactly when and how we will use our specialized skills throughout the project lifecycle.

## User Review Required
> [!IMPORTANT]
> Please review this workflow to ensure the phases align with your vision. Let me know if you would like to add any specific features (like an e-commerce checkout vs. a simple inquiry form) before we move into the Design phase.

## Phase 1: Discovery & Strategy

**Goal:** Define the structure, content needs, and brand narrative before any code or design is written.

* **Step 1: Brand & Content Strategy**
  * **Skill:** `agency-brand-guardian` & `agency-content-creator`
  * **How:** We will analyze the existing Linktree, Instagram, and TikTok to extract the brand's tone of voice. We will draft the core copy: the Homepage headline, the "About Tayjah" story, and the "Custom Orders" descriptions.
* **Step 2: UX Planning & Architecture**
  * **Skill:** `agency-ux-architect`
  * **How:** Map out the sitemap (e.g., Home, Gallery, Menu/Flavors, Order Form, Contact). We will decide on the user journey, ensuring that navigating from a social media link directly to placing a custom cake order is as frictionless as possible.

## Phase 2: Visual Design

**Goal:** Create a mouth-watering, premium aesthetic that matches the quality of the baked goods.

* **Step 3: UI & Visual System Design**
  * **Skill:** `agency-ui-designer` & `agency-visual-storyteller`
  * **How:** The UI Designer will establish the visual tokens—a warm, appetizing color palette, modern typography, and button styles. The Visual Storyteller will curate the best photos and videos from the Instagram and TikTok accounts to use as hero images and gallery grids, ensuring the products look irresistible.

## Phase 3: Development & Implementation

**Goal:** Turn the visual designs into a fast, responsive, and functional website.

* **Step 4: Frontend Foundation & Component Build**
  * **Skill:** `agency-frontend-developer`
  * **How:** We will scaffold a modern web application (using HTML/JS/CSS or a lightweight framework like Vite/React). The developer will build reusable components based on the UI design (e.g., responsive navigation, image carousels for cakes, styling for the order form).
* **Step 5: Social Media & Contact Integration**
  * **Skill:** `agency-frontend-developer`
  * **How:** Integrate the contact/order form (perhaps connecting it to email or a Google Form if keeping it simple). Embed links or feeds to Instagram and TikTok so the site stays visually updated with Tayjah's latest creations.

## Phase 4: Optimization & Launch Preparation

**Goal:** Ensure the site is discoverable and performs flawlessly on all devices.

* **Step 6: Local SEO Optimization**
  * **Skill:** `agency-seo-specialist`
  * **How:** Implement on-page SEO best practices. We will optimize meta titles, descriptions, and header tags for local search terms like "custom cakes in Manchester (M14)", "M14 bakery", and "Treats By Tayjah" to ensure high visibility on Google.
* **Step 7: Social Strategy Alignment**
  * **Skill:** `agency-instagram-curator` & `agency-tiktok-strategist`
  * **How:** Plan the launch strategy for social media. We will advise on how to update the Linktree, IG Bio, and TikTok profiles to drive traffic to the new high-converting website.
